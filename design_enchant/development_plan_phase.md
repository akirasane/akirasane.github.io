## 1. ⌨️ เฟส 1: โครงสร้างเชลล์เทอร์มินัลและคีย์บอร์ด (Core Terminal Shell & Input Handling)

### 📌 เป้าหมาย (Goal)
จัดทำโครงสร้างรากฐานของ Terminal Shell, ระบบเราเตอร์คำสั่ง และการตรวจจับอินพุตคีย์บอร์ดระดับลึก

### 📋 ขั้นตอนการตรวจสอบและผลงานที่ต้องส่งมอบ (Checks & Deliverables)
- [ ] **1. Command Tokenizer & Registry:** เขียนโค้ดกลั่นกรองคำสั่งและจดทะเบียนชุดคำสั่งในโครงสร้าง Router Array
- [ ] **2. Tab-Completion Module:** ทำโมดูลตรวจสกัดคำสั่งและระบบแนะนำเมื่อผู้ใช้พิมพ์ปุ่ม Tab
- [ ] **3. Up/Down History Stack:** ทำประวัติจัดเก็บคำสั่งพิมพ์ย้อนหลัง 50 คำสั่งล่าสุด
- [ ] **4. Autoscrolling Container:** เขียนระบบดีไซน์กล่องเลื่อนสกรอลล์อัตโนมัติ เพื่อให้แถวล่าสุดแสดงล่างสุดของเฟรมเสมอ

---

## 🎨 เฟส 2: เอฟเฟกต์สีสันธีมและหน้าจอเรโทร (Theme Engine & CRT Visuals)

### 📌 เป้าหมาย (Goal)
ตกแต่งหน้ารูปแบบสีสันหลากหลายธีม และวางระบบเอฟเฟกต์ภาพ CRT ที่ดึงอารมณ์ยุคเก่าได้อย่างเป็นธรรมชาติ

### 📋 ขั้นตอนการตรวจสอบและผลงานที่ต้องส่งมอบ (Checks & Deliverables)
- [ ] **1. CSS Variables Switcher:** เขียนฟังก์ชัน `setTheme` ปรับแต่ง CSS Variable เพื่อสลับธีมสีได้ทันที
- [ ] **2. CRT Scanlines Overlay:** พัฒนาแถบเงาพาดขวาง (Scan lines) แบบกึ่งโปร่งใส
- [ ] **3. Subtle CRT Flicker Effect:** เขียนแอนิเมชันความสว่างกระเพื่อมบางเบา พร้อมระบบปุ่มปิดความสว่างหากผู้ใช้ต้องการความนิ่งสบายสายตา
- [ ] **4. Mode Shatter/Rain Transition:** พัฒนาการสลับฉากเปลี่ยนโหมดการแสดงผลในรูปแบบ code falling ฝนรหัสดิ่งตกใส่หน้าจอ

---

## 🎮 เฟส 3: มินิเกมและการตอบโต้อีสเตอร์เอ้ก (Interactive Games & Easter Eggs)

### 📌 เป้าหมาย (Goal)
ผนวกความบันเทิงสอดแทรกเข้าไปในเทอร์มินัลด้วยเกมเล่นสนุกและอีสเตอร์เอ้กที่ตอบรับกับคำสั่งประเภทต่างๆ

### 📋 ขั้นตอนการตรวจสอบและผลงานที่ต้องส่งมอบ (Checks & Deliverables)
- [ ] **1. Snake Canvas Setup:** ติดตั้งงูควบคุมทิศทางผ่านหน้า Canvas 400x400 พิกเซลในหน้าต่างเทอร์มินัล
- [ ] **2. Typing Game Engine:** เขียนโครงสร้างจับคำพิมพ์ถูก/ผิด และวัดผลการ WPM/Accuracy
- [ ] **3. Cowsay & ASCII Portrait Generator:** ติดตั้งอาร์ตวัวพูดได้และภาพประตูปิดท้าย Neofetch ในรูปแบบ ASCII Text
- [ ] **4. Keyboard Focus Release System:** ออกแบบระบบตัดการพิมพ์หลักเมื่อเปิดเกม หรือกดปุ่ม `Esc` เพื่อรีเซ็ตกลับสู่สถานะเชลล์เดิมอย่างปลอดภัย

## 📱 เฟส 4: โหมดพอร์ตกราฟิกการ์ดและตรวจ SEO (GUI Card Layout & SEO/Mobile Tuning)

### 📌 เป้าหมาย (Goal)
สร้างหน้าตาการ์ดพอร์ตสำหรับผู้ใช้งานทั่วไป พร้อมปรับปรุงการทำงานบนสมาร์ตโฟนและคะแนนการค้นหา SEO ของ Google

### 📋 ขั้นตอนการตรวจสอบและผลงานที่ต้องส่งมอบ (Checks & Deliverables)
- [ ] **1. Modern Glassmorphism Cards:** พัฒนาคอมโพเนนต์ GUI Card แสดงผลแยกย่อยตามหมวดหมู่ทักษะ
- [ ] **2. Mobile Terminal Pad:** ติดตั้งชุดปุ่มลัดเสมือนบนมือถือ (Tab, Arrows, Esc, Clear) เพื่อพิมพ์บนเบราว์เซอร์สมาร์ตโฟนได้ราบรื่น
- [ ] **3. SEO & Semantic Structure Auditing:** ตรวจทานความถูกต้องของ HTML5 Semantic Tags, XML sitemap และ meta-description
- [ ] **4. Accessibility & Contrast Verification:** ทดสอบโทนสีและอัตราส่วนความต่างให้อ่านง่าย ผ่านเกณฑ์เครื่องอ่านความปลอดภัย (a11y check) ของเบราว์เซอร์ชั้นนำ
