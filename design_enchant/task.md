# รายการงานและสถานะการตรวจสอบ (CLI Portfolio Checklist)
## ระบบพอร์ตโฟลิโอสไตล์โปรแกรมเมอร์ (Programmer-Style CLI Portfolio)

ใช้สำหรับติดตามสถานะการพัฒนาและการทวนสอบพอร์ตโฟลิโอเทอร์มินัลทีละขั้นตอน

---

### 1. ⌨️ ขั้นตอนที่ 1: แกนระบบและกระบวนการจัดการเชลล์ (Core Shell & Input Engine)
- [ ] **1.1 ระบบถอดและวิเคราะห์คำสั่ง (Parser Setup)**
  - [ ] พัฒนาฟังก์ชันแยกแยะชุดคำและ Argument ของคำสั่ง (Tokenizer)
  - [ ] จัดทำระบบลงทะเบียนคลาสคำสั่งหลัก (Command Registry)
- [ ] **1.2 การคีย์และแป้นพิมพ์ (Interactivity Check)**
  - [ ] ระบบสะกดคำสั่งอัตโนมัติเมื่อกด `Tab` (Tab Completion)
  - [ ] ระบบดึงคำสั่งเก่ามาใช้ใหม่ด้วยปุ่มลูกศร `Up` / `Down` (Command History Stack)
  - [ ] ตัวดักจับปุ่ม `Ctrl + C` เพื่อล้างบรรทัดคำสั่งปัจจุบัน
- [ ] **1.3 ส่วนแสดงผลเทอร์มินัล (UI Window Frame)**
  - [ ] ออกแบบกรอบหน้าต่างแนว Glassmorphism พร้อมปุ่มควบคุมวินโดวส์ 3 สี
  - [ ] ตั้งค่าการแสดงผลแบบอักษร JetBrains Mono หรือ Fira Code
  - [ ] ตัวเลื่อนสกรอลล์อัตโนมัติ (Autoscroll) เมื่อแถวเต็มหน้าจอ

---

### 2. 🎨 ขั้นตอนที่ 2: ระบบจัดการสีและเอฟเฟกต์การเปลี่ยนโหมด (Theme & Transition Effects)
- [ ] **2.1 ชุดตัวแปรสีธีม (CSS Custom Variables)**
  - [ ] เขียนและทดสอบคลาสสลับสีสำหรับธีม Matrix (สีเขียวนีออน)
  - [ ] เขียนและทดสอบคลาสสลับสีสำหรับธีม Dracula (สีม่วงชมพู)
  - [ ] เขียนและทดสอบคลาสสลับสีสำหรับธีม Cyberpunk (สีส้มทองเหลือง)
  - [ ] เขียนและทดสอบคลาสสลับสีสำหรับธีม Nord (สีฟ้าไอซ์แลนด์)
- [ ] **2.2 เอฟเฟกต์จอภาพโบราณ (CRT Effects Overlay)**
  - [ ] ชั้นเอฟเฟกต์ลายเส้นสแกน CRT (Scan Lines)
  - [ ] ระบบแสงกะพริบหน้าจอบางเบา (CRT Flicker) พร้อมตัวเลือกรองรับการเปิด/ปิด
- [ ] **2.3 เอฟเฟกต์การสลับหน้าจอ (Mode Switch Transition)**
  - [ ] พัฒนาแอนิเมชันฝนตกสีเขียวสลับฉากหลัง (Matrix Rain Mode Transition)
  - [ ] ทดสอบประสิทธิภาพเฟรมเรตในช่วงกำลังสลับโหมดไม่ให้กระตุก (50+ FPS)

---

### 3. 🎮 ขั้นตอนที่ 3: ระบบมินิเกมและอีสเตอร์เอ้ก (Games & Easter Eggs Integration)
- [ ] **3.1 ระบบเกมงูสอยผลไม้ (Snake Game)**
  - [ ] ตัวดึงระบบจัดการบน HTML5 Canvas ภายในกล่องเทอร์มินัล
  - [ ] ดักปุ่มคีย์บอร์ดให้ควบคุมงู แทนการพิมพลงหน้าต่างหลัก
  - [ ] ส่วนแสดงผลตารางคะแนนและการปิดเกมด้วยปุ่ม `Esc`
- [ ] **3.2 ระบบเกมวัดความเร็วการพิมพ์ (Typing Speed Game)**
  - [ ] ระบบสุ่มข้อความประโยคคำสั่ง
  - [ ] ตัววัดค่าความเร็วพิมพ์ WPM และระดับความแม่นยำร้อยละ (Accuracy)
- [ ] **3.3 ลิงก์คำสั่งและอีสเตอร์เอ้กขำขัน (Easter Eggs)**
  - [ ] คำสั่งจำลองล้างระบบปฏิบัติการ `sudo rm -rf /` พร้อมข้อความแจ้งบุกรุกเลียนแบบ
  - [ ] ฟังก์ชันคำสั่ง `cowsay <text>` วาด ASCII วัวพูดได้ตามข้อความป้อนเข้า

---

### 4. 📱 ขั้นตอนที่ 4: หน้าการแสดงผลโหมดการ์ดและปุ่มเสมือนบนมือถือ (GUI & Mobile Layout)
- [ ] **4.1 หน้าพอร์ตแบบโมเดิร์นการ์ด (GUI Cards Mode)**
  - [ ] ออกแบบชุดการ์ดกระจก (Glass Cards) สำหรับแสดงประวัติผลงาน ประสบการณ์ และประวัติตนเอง
  - [ ] ระบบตัวสลับโหมด (Switch Toggle Button) ด้านขวาบนของทุกหน้าจอ
- [ ] **4.2 แถบควบคุมปุ่มกดบนสมาร์ตโฟน (Mobile Terminal Pad)**
  - [ ] ออกแบบปุ่มเสมือนสำหรับแตะแทนคีย์บอร์ดจริง: `Tab`, `ArrowUp`, `ArrowDown`, `Esc`, `Clear`
  - [ ] พัฒนาระบบโฟกัสช่อง Input ล่องหนเพื่อบังคับเรียกแป้นคีย์บอร์ดมือถืออัตโนมัติเมื่อแตะส่วนใดๆ บนเทอร์มินัล
- [ ] **4.3 ตรวจสอบด้าน SEO และ AAA Contrast (Accessibility Checks)**
  - [ ] ทดสอบโครงสร้างแท็กให้ตรงตามหลัก SEO (Lighthouse Score > 90)
  - [ ] ตรวจเช็กระดับสีข้อความและพื้นหลังให้ตรงกับความสว่างมาตรฐาน AAA อ่านง่ายไม่ล้าสายตา

---

### 5. 💎 ขั้นตอนที่ 5: การเสริมแต่งหน้าตาและการใช้งานระดับโปร (Advanced UX & Visual Enchantment)
- [ ] **5.1 ปรับปรุงหน้าตาและดีไซน์ IDE Sidebar**
  - [ ] ออกแบบแถบ File Explorer แสดงผลไฟล์จำลอง VFS ฝั่งซ้าย
  - [ ] พัฒนาระบบพับ/ขยาย Sidebar อัตโนมัติ (เปิดค้างบนเดสก์ท็อป ซ่อนบนมือถือ)
- [ ] **5.2 พัฒนาระบบคลิกนำทาง (Click-to-Navigate)**
  - [ ] ตรวจสอบการกดลิงก์ไฟล์ใน VFS Sidebar เพื่อส่งคำสั่ง `cat` หรือ `cd` ไปรันที่ Shell อัตโนมัติ
- [ ] **5.3 ปุ่มทางลัดและดีไซน์ส่วนหัวใหม่ (Quick Shortcuts & Header)**
  - [ ] เพิ่มปุ่มทางลัดคำสั่งหลัก (Badges) เพื่อให้คลิกรันได้ทันที
  - [ ] ออกแบบปุ่มสลับโหมด GUI (Glassmorphic Toggle Button) สีเด่นชัดบริเวณมุมขวาบน
- [ ] **5.4 เพิ่มป้าย ASCII Art บูตระบบ (ASCII Boot Banner)**
  - [ ] แทรกตัวอักษรอาร์ตขนาดใหญ่คำว่า "CHATKAWIN" ลงในเฟสการบูตเพื่อความเป็นมืออาชีพ
