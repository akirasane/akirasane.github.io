# Favicon Files

This directory contains favicon files for the portfolio website.

## Files

- `favicon.svg` - SVG favicon (modern browsers)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `apple-touch-icon.png` - 180x180 PNG for Apple devices

## Generating Favicons

The current PNG files are placeholders. To generate proper favicon files:

1. Visit [RealFaviconGenerator](https://realfavicongenerator.net/)
2. Upload your logo or design
3. Customize the favicon for different platforms
4. Download the generated files
5. Replace the placeholder files in this directory

## Requirements

- **16x16**: Standard favicon size for browser tabs
- **32x32**: High-resolution favicon for modern displays
- **180x180**: Apple touch icon for iOS home screen

All PNG files should be optimized for web use.
